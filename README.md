A PHP library for integrating with [Katapult](https://katapult.io/)

## Version constraints
* V1.x supports PHP 5.6
* V2.x supports PHP 7.4
* V3.x supports PHP 8.1

## Installation

You can install the package via Composer:

```bash
composer require krystal/katapult
```
